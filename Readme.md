#### 程序功能
（1）对于输入中形如/*......*/这样的注释内容需要过滤掉。
（2）如果发现词法或语法错误，输出相应的错误信息。
（3）如果输入是一个合法的语法成分：
  ① 输出以下提示信息：“输入正确，没有词法或语法错误”；
  ② 输出该语法成分对应的语法分析树，采用缩进的文本表示形式。
  
  -----
#### 使用方法
运行后输入PL/0 源程序文件名
回答是否输出虚拟机代码
回答是否输出名字表
fa.tmp 输出虚拟机代码
fa1.tmp  输出源文件及其各行对应的首地址
fa2.tmp  输出结果 
fas.tmp  输出名字表
